# Facebook Signup Page Clone

## 📌 Project Description

This project is a **Facebook Signup Page Clone** built using **HTML, CSS, and JavaScript**.
It replicates the design of Facebook's signup interface and includes **form validation and interactive features**.

The goal of this project is to practice **front-end web development** and understand how real-world signup forms work with client-side validation.

---

## 🚀 Features

* Facebook-style **signup page UI**
* Responsive layout using **CSS Flexbox**
* **Form validation using JavaScript**
* Email format validation
* Password length validation
* Date of birth selection
* Gender selection validation
* Success alert message
* Welcome screen with **Facebook logo after successful signup**

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Form validation and interactivity

---

## ⚙️ How to Run the Project

1. Download or clone the repository

```
git clone: https://github.com/ashrithavarmaa661/facebook_loginpage
```

2. Open the project folder.

3. Double click **page.html** or open it in a browser.

The Facebook signup page will appear.
